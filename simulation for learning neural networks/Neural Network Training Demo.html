🧠 Complete Backpropagation Algorithm Explained

This page demonstrates the complete backpropagation algorithm - the fundamental method neural networks use to learn from data. Here's the full process:

1. The Dataset: We're training the network to learn XOR logic - it should output 1 when inputs are different, and 0 when they're the same. This is a classic problem that requires a hidden layer to solve.

2. Forward Propagation: Input values flow through the network. At each connection, the input is multiplied by the weight. Each neuron sums all its inputs, adds a bias term, and applies the sigmoid activation function: σ(x) = 1/(1+e^-x)

3. Error Calculation (Loss Function): The network's prediction is compared to the target value to calculate the error: Error = Target - Predicted Output. This tells us how wrong the network was.

4. Backpropagation (The Key Innovation): The error is propagated backward through the network to calculate gradients. For each weight, we compute how much it contributed to the error using the chain rule from calculus:
• Output layer: δ_output = error × σ'(output)
• Hidden layer: δ_hidden = (δ_output × weight) × σ'(hidden_value)
Where σ'(x) = x(1-x) is the derivative of the sigmoid function.

5. Gradient Descent (Weight Update): Each weight is adjusted in the direction that reduces the error:
• new_weight = old_weight + learning_rate × δ × input_to_that_weight
Watch the weights change color (green for positive, red for negative) and thickness as they optimize!

6. Iteration & Convergence: Steps 2-5 repeat for each training sample. After many epochs (complete passes through the dataset), the weights stabilize at values that correctly solve the XOR problem. The loss decreases and predictions become accurate.

This is the complete backpropagation algorithm used in all neural networks, from simple networks like this to massive deep learning models. The same principles scale up with more layers, more neurons, and millions of weights!
