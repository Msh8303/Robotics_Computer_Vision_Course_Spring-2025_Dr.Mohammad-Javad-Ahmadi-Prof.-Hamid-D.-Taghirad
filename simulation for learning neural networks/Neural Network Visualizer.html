🔬 How Neural Networks Learn

1. Forward Propagation
Data flows from input neurons through hidden layers to output neurons. Each connection multiplies the input by its weight, and neurons sum these values before applying an activation function. The activation function determines how the neuron "fires" - introducing non-linearity that allows the network to learn complex patterns.
2. Loss Calculation
The network's predictions are compared to the correct answers using a loss function. Higher loss means worse predictions. The goal of training is to minimize this loss.
3. Backpropagation
The error is propagated backward through the network. Using calculus (gradient descent), the network calculates how much each weight contributed to the error.
4. Weight Updates
Weights are adjusted in the direction that reduces error. The learning rate controls how big these adjustments are. Smaller learning rates are more precise but slower; larger rates are faster but risk overshooting.
5. Iteration (Epochs)
This process repeats many times. Each complete pass through the training data is called an epoch. Over time, the network learns patterns and improves its predictions.
